========================================================================
                      Auto Hub - Drive Your Dream Today
========================================================================

Auto Hub is a premium multi-brand digital showroom web application designed 
to showcase a wide variety of automobiles—from electric vehicles to rugged 
SUVs, luxury sedans, and high-performance sports cars.

Live Site:
[Click Here]("https://yuviacc.github.io/aipj/")

------------------------------------------------------------------------
Features
------------------------------------------------------------------------
1. Sleek Navigation: 
   Smooth scrolling, desktop sidebar, and hamburger menu navigation for 
   mobile devices.
2. Premium Theme Toggle:
   Allows switching between the original dark blue/immersive theme and 
   a crisp, modern slate-white light theme. Stored in localStorage for 
   persisted preferences.
3. Car Fleet Directory:
   Filterable catalog of car types (SUV, Sedan, Electric, Sports, etc.) 
   with horsepower, mileage, seating, and estimated price ranges.
4. Booking System:
   Interactive, client-validated booking form for reserving test drives, 
   complete with budget preferences, preferred date & slots, and add-on services.
5. Market Trajectory Charts:
   Live SVG market share breakdown and EV sales growth chart animation.
6. Responsive Layout:
   Fully optimized for all device viewports (Desktop, Tablet, and Mobile).

------------------------------------------------------------------------
How to Run Locally
------------------------------------------------------------------------
Simply open the `index.html` file in any modern web browser.
Alternatively, use a local server like "Live Server" in VS Code to run it at localhost.
